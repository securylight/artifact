# Experimental Setup

The experiments reported in the paper were conducted using the following environment:

- Google Cloud a2-highgpu-1g instance
- 12 vCPUs
- 85 GB RAM
- 1 NVIDIA A100 GPU with 40 GB memory
- 200 GB SSD
- Ubuntu 22.04 LTS

This configuration documents the environment used for the reported results. It is provided to support reproducibility and contextualize the reported training and evaluation workflow.
